# Sistema de Hotel

Este é um sistema de gerenciamento de hotel desenvolvido como trabalho acadêmico.

## O que o sistema faz

- Login
- Cadastro de hóspedes
- Cadastro de quartos
- Cadastro de reservas
- Pesquisa de reservas
- Atualização de registros
- Exclusão de registros
- Check-In
- Check-Out
- Cálculo automático do valor da hospedagem

## Tecnologias utilizadas

- Java
- JavaFX
- PostgreSQL
- JDBC
- Maven
